# Ubiquitous Language (Скрізна мова)

Глосарій термінів проєкту. Ці назви мають використовуватися в коді (класи, змінні) та в спілкуванні.

1.  **Transaction (Транзакція/Витрата):** Одиничний запис про витрату коштів. Має суму, дату та опис. Не може існувати без прив'язки до Користувача.
2.  **Amount (Сума):** Числове значення витрати. Має бути > 0. У цій версії валюта завжди UAH.
3.  **Description (Опис):** Короткий текстовий коментар до витрати (наприклад, "Кава"). Не може бути порожнім.
4.  **Transaction Date (Дата транзакції):** Дата, коли фактично відбулася витрата. За замовчуванням — "сьогодні".
5.  **Category (Категорія):** Мітка для групування витрат (наприклад, "Їжа", "Транспорт").
